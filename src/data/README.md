# Data

This folder contains pairwise comparison matrices used as benchmarks in 
the dissertation research.

## Sources

Expert matrices are drawn from published AHP studies in the literature, 
cited in the dissertation bibliography. No proprietary or confidential 
data is used.

## sample_matrices.json

Contains example 3x3 and 4x4 pairwise comparison matrices demonstrating 
both consistent (CR < 0.1) and inconsistent (CR >= 0.1) judgements, used 
for methodology validation and unit testing.
